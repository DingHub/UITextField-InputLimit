# Extensions of UITextField to forbid input
几个UITextField扩展，用来限制输入
---
eg: forbid emoji, only money decimal , limit length.<br>
如：禁止表情符， 限制金额格式， 限制长度。<br>

Usage:
---
You can use these extensions with code:<br>
可以纯代码使用：<br>

```
textField.isMoney = true
```

Or in IB:<br>
或者简单地在IB里设置：<br>
